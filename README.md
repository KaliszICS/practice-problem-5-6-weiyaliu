# Instructions  

1. Create a function called _getAge()_, which takes in a String array (names), integer array (ages) and String (name) as parameters.
Names and ages are parallel arrays.  Return the age associated with the name.

2. Create a function called _createLengthArray()_, which take in a String array.
Return an integer array with the length of each of the strings.

3. Create a function called _swapPeople()_, which take in a String Array (names), integer array (ages), String (name), String(name).
Swap the two people and their ages in each array.
